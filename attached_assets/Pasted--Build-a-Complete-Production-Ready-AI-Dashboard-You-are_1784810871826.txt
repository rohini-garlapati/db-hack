# Build a Complete Production-Ready AI Dashboard

You are a senior full-stack engineer and UI/UX designer.

Build a complete, modern, production-quality web application using:

* React + Vite
* Tailwind CSS
* React Router
* Axios
* Lucide React icons
* Framer Motion for animations

## Objective

Create a responsive AI dashboard that will later connect to an Azure AI Agent.

The Azure agent is NOT ready yet, so build the application using mock responses, but structure everything so I only need to replace the API endpoint later.

---

Create a premium UI inspired by Stripe, Linear, Vercel, Microsoft Copilot, and ChatGPT.

The design should be:

Use:

* Rounded cards

* Smooth animations
* Beautiful typography
* Light theme
* Loading skeletons
* Toast notifications
* Empty states
* Error states

## API Layer

Create an API service.

Example:

services/api.js

All backend calls must go through this file.

Create functions such as:

* sendMessage()
* uploadDocument()
* getAnalytics()
* getHistory()

Use Axios.

The base URL must come from an environment variable.

Example:

VITE_API_BASE_URL

---

## Backend Integration

Use mock data initially.

Design the code so changing only:

VITE_API_BASE_URL

or the endpoint URL is enough to connect to the real Azure backend.

Do NOT hardcode URLs.
